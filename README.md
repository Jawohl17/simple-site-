# Simple Site Template

Адаптивний шаблон веб-сайту з повною підтримкою різних розмірів екранів, починаючи від 375px.

## Особливості

- ✅ Повністю адаптивний дизайн (375px+)
- ✅ Використання CSS змінних (var()) для кольорів, відступів та розмірів
- ✅ Гнучкі вимірювання (rem, em, відсотки) замість фіксованих пікселів
- ✅ Медіа-запити для мобільних, планшетів та десктопів
- ✅ Мобільне меню з анімацією
- ✅ Плавна прокрутка
- ✅ Сучасний та чистий дизайн

## Структура проєкту

```
.
├── index.html      # Головна HTML сторінка
├── styles.css      # CSS стилі з адаптивністю
├── script.js       # JavaScript для інтерактивності
└── README.md       # Документація
```

## Медіа-запити

Шаблон підтримує наступні breakpoints:

- **375px+** - Мобільні пристрої (базова версія)
- **480px+** - Великі мобільні телефони
- **768px+** - Планшети
- **1024px+** - Десктопи
- **1200px+** - Великі десктопи

## Використання

1. Клонуйте репозиторій або завантажте файли
2. Відкрийте `index.html` у браузері
3. Використовуйте інструменти розробника для тестування на різних пристроях

## CSS Змінні

Шаблон використовує CSS змінні для легкого налаштування:

```css
/* Кольори */
--primary-color: #4a90e2;
--secondary-color: #2c3e50;
--text-color: #333333;

/* Відступи */
--spacing-xs: 0.5rem;
--spacing-sm: 1rem;
--spacing-md: 1.5rem;
--spacing-lg: 2rem;

/* Типографіка */
--font-size-base: 1rem;
--font-size-lg: 1.5rem;
--font-size-xl: 2rem;
```

## GitHub Pages

Для публікації на GitHub Pages:

1. Створіть репозиторій на GitHub
2. Завантажте файли проєкту
3. У налаштуваннях репозиторію увімкніть GitHub Pages
4. Виберіть гілку `main` як джерело
